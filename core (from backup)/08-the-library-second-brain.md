# The Library — The Compounding Layer

Execution completes projects.

The Library compounds insight.

Without a compounding layer,
work disappears.

---

## From Output to Insight

When a project concludes,
its insights are not left inside the project note.

They are extracted.  
Refined.  
Clarified.  
Distilled.

The project moves to Archive.  
The insight moves to the Library.

Some ideas enter directly from the Inbox if they represent principle rather than action.

This is how work compounds.

---

## Library Structure

The Library is not a dumping ground.

It contains refined notes only.

No task language.  
No active deadlines.  
No operational clutter.
